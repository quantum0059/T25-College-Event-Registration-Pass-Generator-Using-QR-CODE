# T25 College Event Registration Pass Generator Using QR CODE

A web application for college event registration that generates unique QR code passes for participants. Helps streamline event entry and management by making registration fast, digital, and secure.

**Language composition**: 
- JavaScript (98.7%)
- HTML (1.3%)

## Features

Based on commit history, the key features include:

- **User Registration:** Allows students to register for college events via a streamlined web interface.
- **QR Code Generation:** Automatically creates a unique QR code pass upon successful registration.
- **Backend Logic:** 
  - Handles event registration request processing.
  - Stores participant information securely.
  - Generates QR code payloads and records for verification.
  - (Located in [`/Backend`](https://github.com/harshit-Upadhyay1921/T25-College-Event-Registration-Pass-Generator-Using-QR-CODE/tree/main/Backend))
- **Frontend Interface:** 
  - Enables users to fill event forms and view registration status.
  - Provides immediate QR code pass download/preview after registration.
  - (Located in [`/frontend`](https://github.com/harshit-Upadhyay1921/T25-College-Event-Registration-Pass-Generator-Using-QR-CODE/tree/main/frontend))
- **Pass Validation:** Facilitators can scan QR codes at the event to verify and record attendance.
- **Security Improvements:** Commit history indicates effort in securing user data and improving verification.
- **Performance & Bug Fixes:** Multiple commits address code refinements, bug fixes, and optimization.

> _Note: Results based on the latest 30 commits. More commit details available [here](https://github.com/harshit-Upadhyay1921/T25-College-Event-Registration-Pass-Generator-Using-QR-CODE/commits?per_page=100)._

## Getting Started

1. Clone the repository.
2. Install dependencies for both `Backend` and `frontend`.
3. Start the backend server (API for registration and QR code generation).
4. Launch the frontend application (event registration UI).
5. Register for an event and get your QR code pass!

## Directory Structure

- **Backend/**: Contains server-side code for registration and QR code operations.
- **frontend/**: Holds the web interface for user interaction.
